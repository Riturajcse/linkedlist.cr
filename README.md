Linked List Implementation for Crystal Programming Language

## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  linkedlist:
    github: scriptnull/linkedlist.cr
```

## Usage

```crystal
require "linkedlist"
```
